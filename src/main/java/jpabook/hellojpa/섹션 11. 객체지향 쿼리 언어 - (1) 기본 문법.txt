=== 1. 객체지향 쿼리 언어 소개 ===

    JPA 는 여러 쿼리 방식을 지원합니다
    JPQL
    JPA Criteria
    QueryDSL
    네이티브 SQL
    JDBC API
    JDBC Template
    MyBatis

    실무에서 Spring Data JPA 와 QueryDSL 을 사용하기 위해선
    JPQL 을 제대로 공부해놔야합니다

    JPQL 은 엔티티 객체를 대상으로 쿼리를 작성하는 것이고
    SQL 은 데이터베이스 테이블을 대상으로 쿼리를 작성하는 것입니다
